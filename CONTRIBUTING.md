# Contributing

Thanks for your interest in the OmniFocus MCP Server!

## Bug Reports and Feature Requests

Please use the [issue templates](https://github.com/s-morgan-jeffries/omnifocus-mcp/issues/new/choose) to file bug reports or request features. Include as much detail as you can — reproduction steps, OmniFocus version, and error messages are especially helpful.

## Development

See [docs/guides/DEVELOPMENT.md](docs/guides/DEVELOPMENT.md) for the internal development workflow, branch conventions, and test procedures.
